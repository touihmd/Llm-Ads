## The Five Pillars

| # | Pillar | Description |
|---|--------|-------------|
| 01 | **Cognitive Separation** | The contextual window is architecturally distinct from conversation space. Zero commercial content inside model responses. Ever. |
| 02 | **Explicit Consent** | User activates the window, selects interest categories, can disable at any moment. |
| 03 | **Quality-First Ranking** | Independent Quality Score (SQL) determines position. Financial bid is secondary. Higher quality = lower cost. |
| 04 | **Full Transparency** | Every content item displays source, SQL score, and exact contextual reason for display. |
| 05 | **White Data Only** | Declared interests + anonymous session context only. No behavioral profiling. No cross-session tracking. Data stays in the LLM ecosystem. |

- The **conversation** = cockpit (pure, uncontaminated by commerce)
- The **contextual window** = separate screen (activated with consent, paid subscribers only)
- The **AI** = honest intermediary, never a salesperson

- Subscription alone cannot fund the model race

LLM-ADS resolves this by making **adoption a self-interest**, not a concession:

| Users | Distrust of systems monetizing attention without consent | Full control, transparency, data stays local |
